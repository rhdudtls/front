문자함수 연습

    0         1         2         3 
    0123456789012345678901234567890
    ABCDEFGHIJKLMNOPQRSTUBWXYZabcde
    ABC       BDEF GHIABC JKLMNO  PQR  ABC       BDEF GHI JKLMNO  PQR

length : 객체(문자/열)의 길이

substring(a,b) : 문자열 발췌(a에서 부터 b앞까지)

slice(a,b) : 문자열 (a에서 부터 b까지)

substr()

replace(a,b) : a를 b로 치환

replaceAll(a,b) : a를 b로 모두 치환

toUpperCase() : 대문자로 변환

toLowerCase() : 대문자로 변환

concat(a,b) :a와 b 문자열 결합

trim() :문자열의 양쪽 공백을 무시

trimStart() :문자열의 앞쪽 공백을 무시

trimEnd() :문자열의 뒷쪽 공백을 무시

replaceAll(' ','') : 문자열의 모든 공백을 무시

charAt(*): 해당 *문자만 발췌

charCodeAt(*): 해당 *문자의 아스키 코드가 나옴: 소문자 시작 97

split() : 특정 문자를 기준으로 잘라낸다. 이때 잘라낸 값들은 배열변수에 기억된다.

indexOf() : 문자/열 검색시에 사용: 있으면 위치값, 없으면 -1 반환

lastIndexOf() : (뒤에서부터) 문자/열 검색시에 사용: 있으면 위치값, 없으면 -1 반환

search() : 문자/열 검색시에 사용: 있으면 위치값, 없으면 -1 반환

match() : 문자열 일치 여부 : 있으면 문자열 그대로 반환, 없으면 공백

includes() : 문자열 포함 여부 : 있으면 true 반환, 없으면 false
